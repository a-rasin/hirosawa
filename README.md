# Assignment 3

## Setup
To install all dependencies run
```
npm run install
```

### .env files
Create one and add a DB property that points to your mongodb uri
```
DB=mongodb://127.0.0.1:27017/assignment3
```

## Run the app
To run the application run
```
npm run dev
```

To build everything:
```
npm run build
```

To start in production mode:
```
npm run start
```
